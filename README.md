### Crazy Math
Crazy math game write on react native

### Install & Run

```
git clone https://github.com/Hieunv1996/CrazyMath
cd CrazyMathh
npm install
```

vs Android

```
react-native run-android 
```

vs IOS

```
react-native run-ios
```
